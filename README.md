# ELLI — Frontend Developer Portfolio

Personal portfolio website built with React and Tailwind CSS.

**Live:** [elli-ma.github.io/portfolio](https://elli-ma.github.io/portfolio/)

## Tech Stack

- React 18
- Tailwind CSS 3
- i18next (EN / RU)
- Font Awesome 6
- Google Fonts (Bebas Neue, Space Mono)

## Features

- Dark / Light theme
- Bilingual (English & Russian)
- Responsive design
- Smooth scroll navigation
- Animated SVG decoration

## Sections

- **About** — intro, contacts (email, Telegram, GitHub)
- **Skills** — frontend, advanced tech, backend, design
- **Education** — university degree
- **Portfolio** — 9 projects with descriptions and tech stacks

## Getting Started

```bash
npm install
npm start
```

## Deploy

```bash
npm run deploy
```

Deploys to GitHub Pages via `gh-pages` branch.
